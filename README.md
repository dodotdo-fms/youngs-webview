# Youngs Webview

Work's on Android 4.4.3 higher. Only support on WebView which based on the chromium.

### Get started

```shell
$ npm install -g bower
$ bower install
$ npm install
```

### Build

```shell
$ npm run build
```

> **NOTE**: If 'gulp' command not found, please install 'gulp-cli' by global via ``$ npm i -g gulp-cli``.


### Development

```shell
$ npm run start
```
